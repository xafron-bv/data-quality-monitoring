How to Use These Viewers

  1. Run the appropriate analysis script to generate reports (e.g., python single_sample_multi_field_demo.py)
  2. Click on the viewer that matches your report type
  3. Upload your JSON report file using the drag-and-drop interface
  4. Explore the interactive visualizations and metrics
